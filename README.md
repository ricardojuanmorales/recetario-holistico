## 🎯 Descripción

**Recetario Holístico** es una plataforma web educativa que conecta **tradición, ciencia y cultura** para democratizar el conocimiento de la gastronomía medicinal. Cada receta incluye análisis fitoquímico, sinergias documentadas y contexto cultural, integrando tres sistemas médicos ancestrales con evidencia científica moderna.

### 🌟 Propósito

Crear un puente verificable entre:
- **Medicina ancestral** (Ayurveda, Medicina Tradicional China)
- **Ciencia moderna** (fitoquímica, farmacología nutricional)
- **Preservación cultural** (custodios del conocimiento, rituales tradicionales)
- **Educación accesible** (experimentos prácticos, objetivos multinivel)

### 👥 Audiencias

- Profesionales de salud holística y nutrición
- Estudiantes de medicina, nutrición y farmacología
- Educadores y docentes
- Entusiastas de gastronomía consciente
- Personas interesadas en medicina preventiva

### 📖 Sistema de 4 Vistas por Receta

Cada receta se presenta desde 4 perspectivas complementarias:

1. **🍲 Receta** - Ingredientes con cantidades exactas, pasos detallados con tiempos y señales visuales
2. **🔬 Ciencia** - Fitoquímica cuantificada, sinergias documentadas, mecanismos de acción
3. **🌍 Cultura** - Historia, origen, custodios del conocimiento citados con nombres y ubicaciones
4. **🎓 Educación** - Objetivos de aprendizaje multinivel, experimentos replicables

### 🌿 Catálogo de Ingredientes

- **50 ingredientes** catalogados con perfiles holísticos completos
- **80+ compuestos fitoquímicos** con cantidades exactas (mg, %, μg)
- **60+ sinergias** documentadas entre ingredientes
- **14 categorías** organizadas (tubérculos, especias, hierbas, etc.)
- **3 sistemas médicos** por ingrediente (Ayurveda, MTC, Occidental)

### 🔍 Funcionalidades Interactivas

- ✅ Búsqueda en tiempo real (ingredientes, compuestos, propiedades)
- ✅ Filtros combinables (categoría, sistema médico, uso terapéutico)
- ✅ Modales fullscreen con navegación entre recetas
- ✅ Tabs interactivos para diferentes vistas
- ✅ Sistema de navegación por capas
- ✅ Diseño responsive (móvil, tablet, desktop)

## 📖 Uso

### Navegación Básica

1. **Explorar por Capas**
   - Usar botones en header: Descubrimiento | Biblioteca | Ingredientes
   - Cada capa ofrece diferente perspectiva del contenido

2. **Ver Recetas**
   - Click en cualquier card de receta
   - Navegar entre tabs: Receta | Ciencia | Cultura | Educación
   - Usar flechas ← → para ver siguiente/anterior receta

3. **Buscar Ingredientes**
   - Ir a capa "Ingredientes"
   - Usar barra de búsqueda (busca por nombre, compuesto, propiedad)
   - Filtrar por categoría o sistema médico

### Ejemplos de Búsqueda

```
"jengibre"           → Encuentra ingrediente por nombre
"curcumina"          → Encuentra por compuesto fitoquímico
"antiinflamatorio"   → Encuentra por propiedad medicinal
"omega-3"            → Encuentra ingredientes ricos en omega-3
```

### Atajos de Teclado (futuro)

```
Esc       Cerrar modal actual
←  →      Navegar entre recetas
/         Focus en búsqueda
1, 2, 3   Cambiar entre capas
```

## 🤖 Desarrollo con IA

### Modelo Utilizado

Este proyecto fue desarrollado con asistencia de **Claude AI** (Anthropic):

- **Modelo:** Claude 3.5 Sonnet
- **Fecha de desarrollo:** Marzo 2026
- **Versión de Claude:** API v1

### Rol de la IA en el Proyecto

#### 1. Arquitectura y Diseño (80% IA)
- ✅ Diseño de sistema de 5 capas navegables
- ✅ Estructura de datos para recetas e ingredientes
- ✅ Modelo de 4 vistas por receta
- ✅ Design System completo (paleta, tipografía, componentes)

#### 2. Desarrollo de Código (90% IA)
- ✅ HTML semántico completo (~4,000 líneas)
- ✅ CSS con variables y sistema de diseño (~800 líneas)
- ✅ JavaScript vanilla para interactividad (~1,200 líneas)
- ✅ Sistema de navegación y modales
- ✅ Búsqueda y filtros en tiempo real

#### 3. Contenido Científico (60% IA + 40% Validación Humana)
- ✅ Investigación de compuestos fitoquímicos
- ✅ Documentación de sinergias entre ingredientes
- ✅ Cantidades exactas de compuestos (mg, %, μg)
- ⚠️ **Nota:** Datos científicos requieren validación con fuentes primarias

#### 4. Contenido Cultural (50% IA + 50% Humano)
- ✅ Investigación de tradiciones culinarias
- ✅ Contexto histórico de recetas
- ⚠️ Custodios del conocimiento: **ficticios** (nombres ejemplares)
- ⚠️ **Requiere:** Entrevistas reales con herbolarios tradicionales

#### 5. Documentación (95% IA)
- ✅ README.md y documentación técnica
- ✅ Changelog y guías de uso
- ✅ Design System exportable
- ✅ Briefs para proyectos relacionados

### Workflow de Desarrollo con IA

```
1. DISEÑO CONCEPTUAL (Humano)
   ├─ Definir visión y objetivos
   ├─ Identificar audiencias
   └─ Establecer principios holísticos

2. ARQUITECTURA (IA asistida)
   ├─ Proponer estructura de capas
   ├─ Diseñar modelo de datos
   ├─ Crear Design System
   └─ Iterar con feedback humano

3. IMPLEMENTACIÓN (IA + Supervisión)
   ├─ Generar código base
   ├─ Implementar funcionalidades
   ├─ Integrar datos científicos
   └─ Validar resultados humano

4. REFINAMIENTO (Colaborativo)
   ├─ Testing de funcionalidad
   ├─ Ajustes de UX
   ├─ Validación científica
   └─ Pulido final
```

### Ventajas del Desarrollo con IA

✅ **Velocidad:** MVP funcional en ~4 horas vs semanas tradicionales  
✅ **Consistencia:** Design System coherente desde el inicio  
✅ **Documentación:** Generación automática de docs exhaustivas  
✅ **Investigación:** Compilación rápida de datos científicos  
✅ **Iteración:** Cambios rápidos sin refactoring manual  

### Limitaciones y Consideraciones

⚠️ **Validación necesaria:** Datos científicos deben verificarse con fuentes primarias  
⚠️ **Contenido cultural:** Custodios ficticios deben reemplazarse con entrevistas reales  
⚠️ **Testing humano:** IA no reemplaza testing real con usuarios  
⚠️ **Contexto ético:** Preservación cultural requiere participación comunitaria  

### Transparencia en el Uso de IA

Este proyecto es transparente sobre el uso de IA:
- ✅ IA como herramienta de desarrollo, no como fuente de verdad absoluta
- ✅ Reconocimiento explícito del rol de Claude AI
- ✅ Compromiso de validar datos científicos
- ✅ Intención de involucrar custodios reales del conocimiento
